Code

Putting this code on your page will render the webring widget:

Webringu PREV RANDOM NEXT

You need to change the slug (see form below).

Please put the code on your website before filling out the form below. It will say "This site isn't part of the webring yet." until I've added you.

You may style it however you want. You can remove the Random link, but please don't remove the Previous or Next links.

Want to join?

Rules

  • Your website should have at least some kind of resources for other webmasters, such as coding tutorials, scripts, link lists (bookmarks), graphic collections, layouts, ...
  • Your website should be coded by you. (No social media profiles, carrds, corporate websites, AI-generated sites, etc)
  • Your website should be (at least mostly) finished. This means you should have at least 2 subpages with content. Homepages with no working links in the navigation do not count as finished.
  • Your website can be hosted anywhere; it does not have to be hosted on Neocities.
  • Your website may not have hate content or discrimination of any kind. Homophobes, transphobes, racists, sexists etc are not welcome.
  • This is not a hard rule, but please make an effort to make your website responsive and accessible.

Form

To join, leave a comment below with your name (whatever you want to be called; NOT the title of your website) and URL to the page with the webring code (e.g. your homepage, links page, ...). In the message, please include which kinds of web resources you have on your website, for example...:

  • coding tutorials
  • layouts
  • bookmarks/links
  • scripts
  • code snippets
  • widgets
  • buttons/blinkies/userboxes
  • graphics/pixels/deco
  • ...

List them in a comma separated list in your preferred order (as you can see in members list above.)

I'll try to add you as soon as possible. Check back to this page or check the webring widget on your website to see if you've been added.
