About the Site

This site was created in February of 2024 after I had a bout of inspiration looking at other Neocities pages. All the design and coding is done by myself. The code for the page is available on GitHub. Feel free to browse around, but please don't steal large chunks of code (or judge me for bad code quality...). For more credits please visit the credits page.

The purpose of this site is to serve me as a creative project as well as a showcase of my work (such as art, playlists, and so on), and all my blathering. Some of the content is taken straight from my listography profile, because that's where I keep most of my stuff online.

The content of this site is ok for teens and up, but I might link to external mature content (such as songs and movies). Some pages of this site might contain triggering content (in the form of gory images), but these have a warning at the top!

The design of this site is a deliberate protest against modern web design trends, because I fricking hate them. Let's make the web fun again!

You can follow me on neocities here.

Accessibility

This site should be usable on any screen size, desktop and mobile, and should work in all browsers. I have made an effort to make the site as accessible as possible.

If you have trouble navigating and using the site, whether you are disabled or not, please feel free to contact me about it!
