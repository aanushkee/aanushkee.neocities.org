My Game Dev Journey

I decided I finally want to learn how to make video games! This page is for anyone who wants to go on that journey with me, or who is just interested!

I am using the free game engine Godot to make a 2D game.

YMMV: Keep in mind that I have some experiencing coding (especially JavaScript and Java). Though I do have to learn a new language for Godot (GDScript), but it's a very easy language to learn! Also, I have a LOT of free time to do stuff like this, and only worked on it on free days, so if you follow along don't feel pressured to do it as the same pace as me!

Day 1

Deciding on a Game Engine

On day 1 I knew I had to decide on a game engine before doing anything else. After researching a bit, I decided to use the free game engine Godot. I chose it because it's supposed to be easy for beginners, can be used to create 2D as well as 3D games, and can export games for PC, Mobile Phones and the Web! It honestly almost seems to good to be true!

Downloading Godot

I downloaded Godot (no installation necessary!) and then downlaoded and took the free Godot Tour to get familiar with Godot's UI. This only took a few minutes and I already felt less overwhelmed after doing it! The step is optional but it's so quick that I do recommend it.


This step is also when I started taking notes. Whenever I thought "Oh, I should remember that" I wrote it down.

Learning GDScript

Now it was time to learn GDScript, Godot's coding language. I read through Learn GDScript From Zero, which is a free interactive tutorial that covers the very basics of GDScript. Most of it was already familiar to me because I have experience coding, and GDScript is not much different than other programming langauges. However, I did learn some valuable Godot-specific things, especially in lessons 7, 10, 11, 15, 21, 24, and 26. I skipped most of the 'practice' parts because they weren't much of a challenge for me. If you're a total beginner to coding I definitely recommend doing them though!

All in all, going through the tutorial took me about an hour. I don't know everything, but I've got the basic syntax down, which is the basis I was looking for. I added some of the new stuff I learned to my notes, so that I wouldn't have to go through the tutorial again.
