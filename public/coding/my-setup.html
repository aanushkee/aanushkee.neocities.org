VSCode Themes

Tip: In the UI, right-click anything you never need to hide it. This will give you more space.

Theme

To change your theme, go to: File -> Preferences -> Theme -> Color Theme and choose one.

I currently use the theme Kimbie Dark which is one of the default themes available in VSCode. It's a dark theme with gentle brown colors. This is what it looks like:

My monitor has some really bad contrast, so in the past I used the theme Saturated Dark+ so I can see the colors better. You can install it like any other extension.

Terminal Theme

To change the colors in your terminal (command line interface), search the settings for "Workbench Color Customizations" and click "Edit in settings.json". In workbench.colorCustomizations add a code form this generator (I use Unikitty Dark). You should see the change immediately.

VSCode Shortcuts

Here are some of the shortcuts I use a lot. Learning these might take a while, but it really speeds up your coding!

  • ctrl + f = search (you can even select text first)
  • ctrl + (spacebar) = display code suggestions (use arrow keys and enter to select one)
  • ctrl + (click) = jump into a function/file (great for JavaScript)
  • ctrl + # = turn current selection into a comment (and reverse)
  • ctrl + d = select the next search result for the current selection (starts multi-cursor)
  • ctrl + alt + â†‘/â†“ = duplicate current line
  • alt + â†‘/â†“ = move current line up/down
  • ctrl + p = opens file search (to quickly open files)

Specific key combinations might differ for you, I don't remember if they're the default...


I also use Emmet (which is built into VSCode) and highly recommend it for quick HTML typing. For example:

  • type ! + press tab key = creates basic HTML file (head, body, etc)
  • type b + press tab key = creates <b></b>
  • type .gallery + press tab key = creates <div class="gallery"></div>
  • type nav.menu + press tab key = creates <nav class="menu"></nav>
  • type lorem + press tab key = adds lorem ipsum (example text):
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Necessitatibus est iusto accusamus omnis eligendi sapiente, porro quibusdam dolorem illum optio dolores ab totam eum eveniet iure et minus minima. Necessitatibus.

VSCode Extensions

You can install extensions in VSCode via the "Extensions" menu. Make sure to restart VSCode after installation.

"Intellisense" generally means code completion and quick info.

"Linters" check for errors.

Here are some extensions I can't do without:

  • Prettier - Code Formatter: automatic formatter for most languages. (note: You don't need an extension for this, I just prefer Prettier)
  • Formatting Toggle: adds a formatting toggle at the bottom right so you can temporarily turn of auto formatting
  • leet-html: syntax highlighting for HTML code in JavaScript files
  • Stylelint: checks for errors in your CSS, SCSS
  • ESLint: checks for errors in your JavaScript (See installation below)
  • SCSS Intellisense: code completion and quick info for SCSS
  • Open in Default Browser: shortcut for opening a current HTML file in your preferred browser (to use: right-click in file, choose 'open in default browser')
  • Highlight Matching Tag: highlights the matching closing HTML tag. (note: VSCode does this by default, but it's not very visible.)
  • Alphabetical Sorter: adds an action to sort highlighted lines alphabetically (to use: ctrl + shift + p + 'sort')

If the automatic formatting for a file is not working, it's probably because the file contains an error! Click the prettier button on the bottom right corner.


How to install ESLint
  1. Install the extension in VSCode
  2. Download a config file from here
  3. Put it into C:\Users\{USERNAME}
  4. In the file, add "browser":true to the list of environments ("env" at the top of the file)
  5. In the file, add "sourceType":"module" to "parserOptions".
  6. In VSCode do ctrl+shift+p "eslint restart server".

Here are some extensions that I don't use but you might enjoy:

  • vscode-pets: adds cute pets! :3
  • footsteps: highlights lines you've edited recently
  • Todo Tree: a list of all your TO DO comments
  • Vibrancy Continued: a neat "glassy" visual effect for the program
  • Rainbow CSV: better color formatting for .csv files
  • SVG Viewer: enabled previews of svg images in VSCode
